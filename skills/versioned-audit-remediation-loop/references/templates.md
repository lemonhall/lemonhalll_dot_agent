# Templates

按需复用这些模板。不要机械照抄，但默认先从这里起步。

## 1. Audit Findings

```md
# YYYY-MM-DD vN Code Audit Findings

## Scope

- 审阅范围：
- 关联文档：
- 关联代码：

## Findings

### Finding 1 标题

- 严重级别：`P0/P1/P2`
- 问题：
- 风险：
- 根因：
- 建议修复方向：
- 关联文件：
```

## 2. Backlog Entry

```md
### Vn-ISSUE-XXX 标题

- **问题**
  - ...
- **为什么是 `P0/P1/P2`**
  - ...
- **状态**
  - todo / in_progress / done
- **需要改的文档**
  - ...
- **文档改法**
  - ...
- **实现思路**
  - ...
- **最低验收**
  - ...
```

## 3. Issue Plan

```md
# Vn Issue XXX Topic Plan

**Goal:** ...

### Task 1: 红测

- Modify: ...

- [ ] Step 1 ...

### Task 2: 实现

**Files:**
- Modify: ...

- [ ] Step 1 ...

### Task 3: 验证

**Files:**
- Test: ...

- [ ] Step 1 ...

### Task 4: 文档回写

**Files:**
- Modify: ...

- [ ] Step 1 ...
```

## 4. Completion Record

```md
- **完成记录**
  - 改了什么
  - 为什么这样改
  - 已新增或更新哪些测试
  - 已验证：
    - `command 1`
    - `command 2`
```
